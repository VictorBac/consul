# Consul-Terraform-Sync Enterprise

<EnterpriseAlert>
  Consul-Terraform-Sync Enterprise features are available with a Consul license from
  all versions of self-managed{' '}
  <a href="https://www.hashicorp.com/products/consul/">Consul Enterprise</a>.
</EnterpriseAlert>

Consul-Terraform-Sync Enterprise features address organization complexities of collaboration, operations, scale, and governance.

Consul-Terraform-Sync Enterprise is a part of [Consul Enterprise](https://www.hashicorp.com/products/consul).
Its enterprise features, listed in the sidebar navigation on the left, require a Consul
[license](/docs/nia/enterprise/license) to be applied.
